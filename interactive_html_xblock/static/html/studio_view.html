Interactive JS Block Editor

Create interactive content with HTML, CSS, and JavaScript. Use submitInteraction(data) in your JavaScript to capture learner interactions.

Enter the HTML content that will be rendered in the interactive block.

Enter CSS styles for your interactive content. Styles will be scoped to this block.

Enter JavaScript code for interactivity. Use submitInteraction(data) to send data to the XBlock.

Example: 
function submitAnswer(answer) {
  submitInteraction({
    answer: answer,
    question: "What is 2 + 2?",
    correct: answer === "4",
    timeSpent: Date.now() - startTime
  });
}

Show debug information and console logs in the student view.

Automatically grade based on interaction data containing 'score', 'grade', or 'correct' fields.
